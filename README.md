# Healthcare Patient-Doctor Platform

## Features
- JWT-based authentication (patient & doctor roles)
- 3-step onboarding with draft saving
- Doctor assignment
- Real-time chat using Socket.io
- PostgreSQL (Supabase) database

## Tech Stack
- Frontend: Angular + Tailwind
- Backend: Node.js, Express
- Realtime: Socket.io
- Database: Supabase (PostgreSQL)

## Setup
1. Clone repo
2. Setup `.env` files
3. Run backend and frontend
4. Import `db/schema.sql` into Supabase
